【使用方法】:

T-gel與T-sal區別：

T-gel: 適用於頭皮屑特別嚴重或者脂溢性皮炎，不建議長期使用的。

T-sal: 適用於頭皮屑不是那麼多，可以長期使用的。

如果頭皮屑真心很嚴重，可以先使用T-gel，待好轉一些之後，長期使用T-sal
